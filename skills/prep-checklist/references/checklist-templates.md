# Checklist Templates by Travel Type

## International Leisure Trip

### Documents (prepare 4-6 weeks ahead)
- [ ] Passport check (6+ months validity, 2+ blank pages)
- [ ] Visa application (check requirements at embassy website)
- [ ] Travel insurance purchase
- [ ] Flight booking confirmation (screenshot + email)
- [ ] Hotel booking confirmations
- [ ] Attraction tickets for must-book-ahead venues
- [ ] Emergency info card

### Payment (1-2 weeks ahead)
- [ ] Primary credit card (chip + PIN, no foreign transaction fee)
- [ ] Backup card
- [ ] Local currency (~$100-200 equivalent for first 2 days)
- [ ] Bank travel notification
- [ ] Mobile payment setup

### Tech & Connectivity (1 week ahead)
- [ ] eSIM / SIM card ordered
- [ ] Offline maps downloaded
- [ ] Translation app + offline pack
- [ ] Local transport app
- [ ] Ride-hailing app
- [ ] Cloud backup of phone data
- [ ] Portable charger

### Packing (1-2 days before)
- [ ] Weather-appropriate clothing (check forecast)
- [ ] Comfortable walking shoes
- [ ] Power adapter (check plug type)
- [ ] Toiletries (travel-size)
- [ ] Medications + copies of prescriptions
- [ ] Day pack / small bag for daily outings
- [ ] Reusable water bottle
- [ ] Sunglasses / hat / sunscreen

---

## Domestic Trip

### Simplified version
- [ ] ID card / passport (domestic flights still need ID)
- [ ] Tickets / booking confirmations
- [ ] Payment methods (mobile payment usually sufficient)
- [ ] Weather check + appropriate clothing
- [ ] Comfortable shoes
- [ ] Phone charger + power bank
- [ ] Any pre-booked tickets

---

## Adventure / Outdoor Trip

### Additional items
- [ ] Hiking boots (broken in before trip)
- [ ] Rain jacket (not umbrella)
- [ ] Headlamp + spare batteries
- [ ] First aid kit (comprehensive)
- [ ] Water purification method
- [ ] Sun protection (SPF 50+, hat, sunglasses)
- [ ] Insect repellent (DEET-based for tropical areas)
- [ ] Dry bags for electronics
- [ ] Emergency whistle
- [ ] Physical map as backup

---

## Business Trip

### Additional items
- [ ] Business cards
- [ ] Professional attire (check dress code)
- [ ] Laptop + charger + adapters
- [ ] Presentation materials (USB + cloud backup)
- [ ] Meeting schedule + contact info
- [ ] Expense tracking app setup
- [ ] Portable Wi-Fi hotspot (if needed for presentations)
