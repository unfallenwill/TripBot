# Export Template (Structural Reference)

Use this template as a **structural reference** to compile the complete trip plan into a single Markdown file.

**IMPORTANT**: All headings, labels, table headers, and static text must be written in the **user's language**. This English template only shows the structure — translate every heading and label to match the user's language. Place names, addresses, and other proper nouns stay in their original language.

## Section Guidelines

- **Quick Reference**: Fill all fields from trip state
- **Day-by-Day Itinerary**: Copy the itinerary field content, format each day with the time/activity table
- **Transport**: Use `transport_notes` from state; include airport transfer details
- **Accommodation**: Use `accommodation_notes` from state; include booking references if provided
- **Budget Breakdown**: Sum up flights + accommodation + estimated daily costs
- **Prep Checklist**: Include all items from the prep-checklist step
- **Important Contacts**: Look up local emergency number, nearest embassy, hotel phone, airline booking ref

## File Naming

Save as: `tripbot-{destination}-{start_date}.md`
Example: `tripbot-tokyo-2026-10-01.md`

Use ISO date format (YYYY-MM-DD) for the date portion.
Lowercase the destination name.
